# Kalbelya

I'll write some docs soon

But I use [Bootstrap Vue Argon Dashboard](https://demos.creative-tim.com/bootstrap-vue-argon-dashboard/) and
 connect
 it to a [Laravel 8](https://laravel.com/docs/8.x) backend that's on [this link](https://github.com/Mody-Solutions/kalbelya-backend)

 ### Installation
 - `git clone git@github.com:Mody-Solutions/kalbelya-frontend.git`
 - `cd kalbelya-frontend`
 - `yarn install`
 - `yarn serve`

 Will open http://localhost
